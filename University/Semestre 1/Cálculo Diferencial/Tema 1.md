## ¿Que es una función?
Es una regla que asigna exactamente un elemento de un conjunto con exactamente otro elemento de otro conjunto.

normalmente se usan las letras f, g o h para nombrar funciones

variable dependiente y variable independiente

algunos tipos de funciones:
* Constante $f(n)=1$
* Logarítmica: $f(n) = log \enspace n$
* Lineal: $f(n) = n$
* Lineal-logarítmica: $f(n)  = n^2$
* Cuadrática: $f(n) = n \enspace log \enspace n$
* Polinómica: $f(n) = n^k$
* Exponencial: $f(n) = 2^n$
* Factorial: $f(n) = n!$
